# Publisher Dev

Base Next.js para criar posts profissionais com IA e automação de publicação.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- tRPC + React Query
- Prisma (PostgreSQL)
- Zustand
- Vitest + Testing Library
- Playwright

## Primeiros passos

```bash
npm install
npm run dev
```

## Scripts úteis

- `npm run lint`
- `npm run test`
- `npm run test:e2e`
- `npm run prisma:generate`
- `npm run prisma:migrate`
